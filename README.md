# 🌐 AI Website Copy Generator for Local Businesses  
### Prompt Engineering Task 1 (2026)

---

## 📌 Project Overview  
This project focuses on using **Prompt Engineering + AI tools** to generate high-quality, conversion-focused website copy for local businesses.

Many small businesses struggle with:
- unclear website messaging  
- weak value propositions  
- poor calls-to-action  

This system solves that by generating **ready-to-use website content** using structured prompts.

---

## 🏢 Business Chosen  
**GlowCare Skin Clinic, Hyderabad**

- Type: Clinic  
- Target Audience: People with skin concerns  
- USP: Advanced dermatology treatments with personalized care  

---

## 🎯 Objective  
To design a reusable prompt framework that can:

- Generate homepage copy  
- Create service descriptions  
- Write persuasive CTA sections  
- Adapt tone based on business type  
- Produce real-world, publish-ready content
